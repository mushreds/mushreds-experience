# Story 3.1: Galeria de Sonhos Sintéticos

## Descrição
Implementar o portfólio imersivo de mídia gerada por IA (Imagens e Vídeos) conforme a arquitetura de dados e visual minimalista.

## Contexto
- Ref: `docs/prd/02-features.md` (Feature 2)
- Ref: `docs/architecture/03-data-models.md` (AI Gallery Item)

## Tarefas
- [x] Criar o componente `GalleryGrid`.
- [x] Desenvolver componente `GalleryItem` para renderizar imagens e vídeos de forma otimizada.
- [x] Implementar vídeos em loop contínuo, silenciados e sem controles (NFR-03).
- [x] Desenvolver o efeito "Hover Mono": Ao passar o mouse, revelar o prompt em texto roxo monoespaçado sobreposto à imagem.
- [x] Adicionar lazy loading de vídeos para não travar o carregamento inicial.

## Critérios de Aceitação
- [ ] Galeria responsiva (grid dinâmico).
- [ ] Vídeos iniciam automaticamente e em loop suave.
- [ ] Prompts aparecem de forma nítida e futurista.
- [ ] Performance Lighthouse > 90 (NFR-02).
