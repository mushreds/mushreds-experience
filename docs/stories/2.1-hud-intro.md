# Story 2.1: Experience Intro HUD (Scanner)

## Descrição
Implementar a animação de abertura futurista que simula um scanner de realidade aumentada, capturando a atenção do usuário vindo do QR Code.

## Contexto
- Ref: `docs/prd/02-features.md` (Feature 1)
- Ref: `docs/architecture/01-system-design.md` (HUD Engine)

## Tarefas
- [x] Criar componente `IntroHUD` com fundo preto absoluto.
- [x] Implementar efeito de "Glitch" inicial em roxo neon via Framer Motion.
- [x] Desenvolver lógica para capturar e exibir metadados fictícios/reais (IP, Localização, Device).
- [x] Criar animação da "Scan Line" horizontal (CSS/Framer Motion).
- [x] Implementar transição de saída (Fade-out) e persistência no LocalStorage para evitar repetição (CON-02).

## Critérios de Aceitação
- [ ] A animação dura entre 2 e 3 segundos.
- [ ] O scan percorre toda a tela mobile/desktop.
- [ ] O site transita suavemente para a Home após o scan.
- [ ] Usuários recorrentes pulam a animação automaticamente.
