# Project Brief: Mushreds Digital Experience

## 1. Visão Geral
- **Cliente**: Mushreds (Agência de Marketing)
- **Localização**: Goiânia, GO (Operação Global)
- **Core Business**: Automações com IA, Geração de Mídia Sintética (Imagens e Vídeos).
- **Objetivo**: Criar uma presença digital impactante, futurista e minimalista que sirva como portfólio vivo de suas capacidades em IA.

## 2. Identidade Visual & Tom
- **Paleta de Cores**: Preto Absoluto (#000000) e Roxo Neon/Elétrico (#8A2BE2 ou similar).
- **Estilo**: Futurista, Minimalista, Abstrato (Estilo Apple/Tesla/Neuralink).
- **Tom de Voz**: Global, sofisticado, tecnológico e autoritário.

## 3. A "Experiência de Entrada" (The Wow Factor)
- **Origem**: QR Code Físico (Muro/Fachada da Mushreds em Goiânia).
- **Conceito**: HUD Reality Scanner (Interface de Realidade Aumentada simulada).
- **Elementos**: 
  - Glitch roxo neon na abertura.
  - Linhas de scan processando a entrada.
  - Dados dinâmicos exibindo localização (Goiânia -> Global) e status da IA ("Generating Assets...").
  - Transição fluida para o site minimalista.

## 4. Seções Principais
### 4.1 Galeria de Sonhos Sintéticos
- Portfólio de mídia gerada por IA (Stable Diffusion, Sora, etc.).
- Integração de vídeos como texturas de fundo ou elementos orgânicos, evitando players convencionais.
- Interação hover que revela o prompt original das criações.

### 4.2 O Painel de Automação
- Infográfico abstrato e animado.
- Visualização de fluxos de dados e conexões de rede neural invisíveis.
- Foco em escala e inteligência operacional.

### 4.3 O Prompt de Soluções (CTA)
- Interface de chat ou campo de prompt minimalista.
- O usuário digita um problema e o sistema sugere uma automação Mushreds.
- Conversão para lead via interface de alta tecnologia.

## 5. Requisitos Técnicos Iniciais
- **Performance**: Carregamento ultra-rápido (essencial para usuários de mobile via QR Code).
- **Animações**: Framer Motion ou Three.js para os efeitos de scan e HUD.
- **Mobile-First**: A experiência via QR Code deve ser perfeita.
